# cp2-final-vibecoding

## Emo-gotchi MVP

本專案目前提供一個 Next.js + React MVP 原型，實作規格書中的核心循環：

- 情緒輸入與本機模擬 AI JSON 分析
- 小怪獸外觀、心情值、負面值、正面值變化
- 治癒任務分配、完成、放棄
- 配件獎勵與頭飾/臉部/身體部位限制
- 怪獸日記封存
- 匿名交流區分享與留言

### Routes

- `/`：情緒餵食與目前怪獸
- `/tasks`：治癒任務
- `/diary`：怪獸日記
- `/community`：匿名交流區

### 執行方式

```bash
npm install
npm run dev
```

開啟：

```text
http://127.0.0.1:3000
```
